## 用CSS实现三角符号

```css
/*记忆口诀：盒子宽高均为零，三面边框皆透明。 */
    div:after {
      position: absolute;
      width: 0px;
      height: 0px;
      content: " ";
      border-right: 10px solid transparent;
      border-top: 10px solid blueviolet;
      border-left: 10px solid transparent;
      border-bottom: 10px solid transparent;
    }
```
